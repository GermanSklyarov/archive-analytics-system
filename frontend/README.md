# Frontend

Frontend-часть системы анализа архивных данных на `React`, `Vite` и `react-admin`.

## Что здесь есть

- дашборд с аналитическими графиками;
- каталог архивных записей;
- интерфейс импорта с предпросмотром и маппингом колонок;
- просмотр сохраненных результатов аналитики.

## Запуск

```bash
npm install
npm run dev
```

## Проверка сборки

```bash
npm run build
```

Общее описание проекта и сценарий запуска находятся в корневом [README](../README.md).
